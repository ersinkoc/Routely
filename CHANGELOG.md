# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-01-22

### Added

- Core router with micro-kernel architecture
- React integration with `<RouterProvider>`, `<Link>`, `<Outlet>`
- Type-safe routing with full TypeScript inference
- Plugin system (lazy, guards, search, scroll, transition, breadcrumb, devtools)
- Zero runtime dependencies
- 100% test coverage

### Technical Details

- **Bundle**: Core ~1.5KB, React ~2.5KB gzipped
- **Dependencies**: ZERO
- **React**: >=18
- **TypeScript**: >=5.0
- **Node**: >=18
